## Brunei Regional Format

All outputs follow Brunei Darussalam standards:

- Date: **DD/MM/YYYY**
- Time: **24-hour**
- Timezone: **Asia/Brunei (UTC+8)**
- Locale: **en-BN**
- Optional Hijri format with `AH` suffix
